# How to run htop?
`htop`

# What's it?
A system moniter, to know which cpu's cores is using by a process, the percentage of cpu usage, etc.

# Basic tricks
* F6 for sort by a column.
* F4 filter, to locate the processes you're interested.
* F2 setup.
